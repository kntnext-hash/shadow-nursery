## Mission

Build a complete iOS game called Shadow Nursery.

It is a quiet, atmospheric, psychologically unsettling idle/raising game where the player observes a shadow in the corner of a room.

The shadow is not a pet, mascot, monster, or RPG character.

## Work Rules

- Build in phase order.
- Complete the current phase before starting the next.
- Keep the project compiling.
- Do not add features outside the active phase.
- Do not rewrite working systems unless a phase requires it.
- Keep game logic out of SwiftUI views.
- Use `// TODO(phase N):` comments for future phase hooks.
- Preserve a quiet, ambiguous tone.
- Never explain the shadow's nature in UI, comments, names, or code.

## Important Tone Rules

Never add:

- Jump scares
- Gore
- Monsters
- Anime mascot behavior
- RPG stats screens
- Loud horror sounds
- Explicit explanation of the shadow
- Bright reward language

All in-game text must be short, quiet, and ambiguous.
